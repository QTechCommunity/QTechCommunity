Mine Chromatic Addon

Adds working devices to the game, including laptops.
NOTE THAT ALPHA AND BETA VERSIONS MAY INCLUDE GAME BREAKING BUGS AND CRASHES

Features

  • Adds a webbrowser app to Ultreon Devices Mod

Notes

This mod:

  • Does not work on Android, or systems other than macOS/Linux/Windows due to the required MCEF.
  • Requires Ultreon Devices Mod and MCEF mods to be installed to work.

Source code

The source code is on GitHub: Ultreon / mine-chromatic-addon
