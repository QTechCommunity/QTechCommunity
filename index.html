Probably just another random things mod

Random Thingz

Probably just another random things mod. This is a mod that adds a bunch of random things, such as a slime sword, earth staff, a magnet that holds RF energy and when active attracts items to the player, a more advanced debug menu, an exit confirmation menu (can be configured and disabled), a bunch of tools and weapons, traits that are on the custom tools (traits are a sort of enchantments, but these can't be changed and are default for the tool / weapon material).

My First Game

Bubble Blaster

The first project I've ever made. A game where you need to pop bubbles for getting points, and getting higher levels how further you go, each bubble has its own functionality and can act different by getting different points per bubble. The bubbles have its own properties like: attack damage; defense; score; effect given, and more. Originally made in Python, first made with tkinter, then using pyglet. And now made in Java with the Java AWT api.

Notepad but better

Notepad Improved

This app has a different method for text editing, it can contain multiple editors in one window. This allows to manage the editors better without having windows spread everywhere on your screen. The way the app does it is by creating virtual windows inside the main window. Also, the app and editor can be customized, for example: foreground and background colors.
