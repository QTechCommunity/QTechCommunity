An Ultreon Original

Bubble Blaster 2

The first project I (Qboi) have ever made. A game where you need to pop bubbles for getting points, and getting higher levels how further you go, each bubble has its own functionality and can act different by getting different points per bubble. The bubbles have its own properties like: attack damage; defense; score; effect given, and more. Originally made in Python, first made with tkinter, then using pyglet. And now made in Java with the Java AWT api. Note: this project is going to be ported to LibGDX too.

A mod that adds working devices to Minecraft

Devices Mod

This mod adds a working laptop, with an custom operating system made for the mod. Featuring a bunch of apps including GitWeb. The mod is a fork of MrCrayfish's Device Mod. Me (@Qboi) was challlenged to port this mod to modern versions of Minecraft. Of course I accepted that challenge and ported it. I got (indirectly) permission of MrCrayfish as he said everyone was allowed to fork it and make their own version. So at some point @Jab125 asked if he can help me. At some point we released it.

Notepad but better

Notepad Improved

This app has a different method for text editing, it can contain multiple editors in one window. This allows to manage the editors better without having windows spread everywhere on your screen. The way the app does it is by creating virtual windows inside the main window. Also, the app and editor can be customized, for example: foreground and background colors.
